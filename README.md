# Personal Portfolio
> version 2.0.0

This repo consists in the development of the second version of my personal portfolio

## Development environment

To setup the development environment, it is necessary to install:
- Ruby
- Jekyll 3.x
- jekyll-sitemap
- gemoji
- jekyll-pagination
- jekyll-paginate
- jekyll-frontmatter-jsonify
